Описание

Сценарий поможет автоматически настроить сервер для CS-Cart и Multi-Vendor 4.0+. Протестировано в Айхоре, FastVPS и Digital Ocean.

Поддерживаемые ОС

  • Ubuntu 14.04 x86_64
  • Ubuntu 14.10 x86_64
  • Ubuntu 15.04 x86_64
  • Debian 6 Squeeze x86_64
  • Debian 7 Wheezy x86_64
  • CentOS 6 x86_64
  • CentOS 7 x86_64

Ограничения

  • Не работает на виртуальных хостингах.
  • Не работает с панелями управления ISPManager, cPanel, Plesk и пр.
  • Нужны права суперпользователя.

Использование

Установите Ansible и его зависимости

cd /tmp
curl -sL https://gist.github.com/gongled/a524d42d4a4ad8551756/raw/ansible-bootstrap.sh | /bin/bash
cd ansible

Исправьте конфиг group_vars/all

stores:
    foo.com:
        storefronts: ['foo.org']
        adminpanel: "secretfoo.php"
        pool: "www"
        root: "/var/www/html/foo.com"
    bar.com:
        storefronts: []
        adminpanel: "securebar.php"
        pool: "www"
        root: "/var/www/html/bar.com"

Исправьте конфиг group_vars/db

Измените пароль root для базы данных.

mysql:
    password: lovesecretphrase

Запустите настройку для установки

LEMP: NGINX, MySQL и PHP

ansible-playbook lemp.yml -c local

Готово.
